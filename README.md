# solid-octo-sniffle
# Simple Quiz Application

A clean, responsive, and interactive multiple-choice Quiz Web Application built using pure HTML5, CSS3, and vanilla JavaScript. There are currently only 14 questions i will expand them in future.

This application dynamically renders general knowledge trivia questions, tracks user selections in real-time, displays immediate color-coded feedback for correct or incorrect answers, and tallies the final score at the end of the session.

---

 Features

Dynamic Question Rendering: Questions and answers are populated dynamically via JavaScript arrays rather than hardcoded in HTML.
Instant Feedback: Instantly reveals the correct answer by highlighting buttons green (correct) or red (incorrect) upon user selection.
Answer Lock-in: Prevents users from changing their minds by disabling all choice buttons after an answer is selected.
Live Score Tracking: Tracks user performance throughout the trivia session.
Replay Functionality:  Users can seamlessly restart the quiz from the final score screen using a "Play Again" loop.
Responsive UI: Styled cleanly with custom Google Fonts (`Poppins`) and optimized layout scaling across desktop and mobile devices.
